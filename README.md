# dotpoint-library
Useful tools, datastructure, helper, etc. 

## prerequisite
Haxe 4.x and a supporting IDE such as Visual Studio Code.
```shell
sudo add-apt-repository ppa:haxe/releases -y
sudo apt-get update
sudo apt-get install haxe -y

mkdir ~/.haxe && haxelib setup ~/.haxe
```

## build
```shell
haxe build-all.hxml
```
